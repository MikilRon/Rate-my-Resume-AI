## Overview 📖

RatemyresumeAI is a modern resume analysis web application designed to simulate professional ATS (Applicant Tracking System) feedback in a clean, startup-style interface.

Users can paste their resume and a job description to receive:

* ATS-style scoring
* skill match analysis
* strengths & weaknesses
* missing keywords
* improved resume bullet suggestions

Built as a portfolio project focused on frontend engineering, UI/UX, and AI-inspired product design.

## Demo Mode 🧪

RatemyresumeAI currently includes a professional demo analysis fallback system.

If live AI analysis is unavailable, the application automatically generates realistic sample resume feedback to maintain a smooth user experience. (currently beta version only supports demo analysis)

This allows:

* uninterrupted demos
* reliable deployment
* frontend/UI testing
* portfolio showcasing without backend dependency

## Installation 🖥️

Clone the repository:

```bash
git clone https://github.com/yourusername/ratemyresumeai.git
```

Navigate into the project:

```bash
cd ratemyresumeai
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
